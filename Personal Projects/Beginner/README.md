## Structure

Each project uses the same starter layout:

- `README.md` for scope and goals
- `src/main.py` as the entry point
- `docs/roadmap.md` for implementation milestones
- `docs/developer_guide.md` for developer-facing explanation

## Developer Guide Scope

Each `developer_guide.md` explains:

- what the project is expected to do
- how the implementation should work
- where the project would realistically be used
- what another developer should keep in mind while building it

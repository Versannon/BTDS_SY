# Mad Libs Generator Developer Guide

## What Is Done

This scaffold defines a template-based word game where users provide words that are inserted into a predefined story.

## How It Is Done

The implementation should maintain one or more text templates with named placeholders. User input is gathered for each placeholder, and the final output is generated using string formatting.

## Where It Will Be Used

It is commonly used for entertainment, classroom exercises, and practice with strings and user input.

## Developer Explanation

Treat templates as data, not hardcoded print blocks. That makes the project easier to extend with multiple stories and potentially file-based template loading.
